# Luderis — Rediseño (prototipo)

Prototipo HTML del rediseño de Luderis. **No es la app de producción** (React/Vite); es una maqueta navegable e interactiva para revisar y compartir.

## Cómo verlo
Abrí `redesign/Luderis - Inicio.html` en el navegador (necesita internet para cargar React/Babel desde CDN).

## Estructura
- `redesign/` — el prototipo (HTML + módulos `.jsx`/`.js` mock).
- `assets/logo.png` — logo referenciado por el prototipo (ruta relativa `../assets/logo.png`).
